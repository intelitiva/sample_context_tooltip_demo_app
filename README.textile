h1. Sample Context Tooltip Demo App

This is a demo application to show how to use the context tooltip plugin

h2. Using

This is a rails application, just clone it, and install the context tooltip plugin:

<pre>
  ./script/plugin install git://github.com/intelitiva/context_tooltip.git
</pre>
